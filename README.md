#electron-skeleton

A bootstrap Electron codebase using Angular 2 and Bootstrap CSS

A 3 paged navigation structure has been included as quick start code

##Install
npm install

##Usage
npm run electron

##Developer

###Build
npm run build

###Watch
npm run watch
